## Lessons-pending (escalated)

This event escalates two existing lessons-pending out of "tracked" into "must-fix before EOD":

1. **CB equity computation source** — must switch to paper_sim cash + paper_sim MTM. Today's run uses this source ($100,655.08) but the persisted peak ($119,140.25) was anchored to the broker-mixed source, producing a structural mismatch. A drafted `prompts/proposed_updates/cb_equity_source.md` would let the next EOD reset the peak to the paper_sim-true level. Until then, OUT state is essentially "stuck" by construction.
2. **CB peak recalibration** — after the equity-source fix, the peak should be recomputed from paper_sim history. Recommended approach: drop the persisted peak when the source flips so the breaker re-anchors on the next observation.

Neither correction can be made by this routine (CB code is `lib/portfolio_risk.py` — `lib/` writes are not in the routine's approved write paths beyond state files, and changing the equity-source contract is a behavioural change that wants a PR review).

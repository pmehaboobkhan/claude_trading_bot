## Schema

```yaml
agent_name: <string>
first_observed: <YYYY-MM-DD>           # date this agent first issued a prediction with confidence
predictions_made: <int>                # count of confidence-tagged outputs
predictions_reconciled: <int>          # count whose 1d/5d/20d windows have closed

  "0.50-0.60":
    predictions: <int>
    hit_rate_1d: <float or null>
    hit_rate_5d: <float or null>

  "0.60-0.70": { ... }
  "0.70-0.80": { ... }
  "0.80+":     { ... }
notes: <string>                        # descriptive only — no verdicts
```

## Hit-rate definition
A prediction is considered a "hit" at horizon `h` if the realized direction matches the predicted direction (long-bias predictions: close[h] > entry; short-bias: close[h] < entry). Magnitude calibration (predicted return vs realized return) is tracked separately once N >= 30.
